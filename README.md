# Keerthirajan Murugesan - Portfolio

A modern, responsive portfolio website built with Next.js, TypeScript, and Tailwind CSS.
Features a dark theme with vibrant AI/ML-inspired accents, smooth scrolling, and responsive design.

### Editing Content
All content (personal info, skills, projects, experience) is stored in a single file for easy editing:

- **Colors**: Defined in `src/app/globals.css` (CSS variables for background, primary, secondary, accent).
- **Icons**: Using [Lucide React](https://lucide.dev/).
